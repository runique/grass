Lua基本句法

让我们开始创建第一个Lua程序

交互模式

Lua提供了交互模式。在交互模式下，你可以在命令行输入程序，回车后得到运行结果。例如：

$ lua -i 
$ Lua 5.1.4  Copyright (C) 1994-2008 Lua.org, PUC-Rio
quit to end; cd, dir and edit also available

输入：

print("test")

默认模式

默认模式下，Lua解释器(intepreter)遍历并执行Lua文件

将Lua程序存为扩展名为.lua的文件。编辑一个文件test.lua，内容为：

print("test")

运行

$ lua test.lua

将得到以下结果：

test
也可隐式的运行Lua解释器。这种情况下，以上的test.lua将加入一行路径指示Lua解释器的路径在哪里，例如： 
#!/usr/local/bin/lua

print("test")

其中#!/usr/local/bin/lua指出，Lua解释器在/usr/local/bin/路径下。这样，通过修改test.lua的运行权限(linux下)，可以在shell执行运行test.lua文件：

$ chmod a+rx test.lua
$./test.lua
输出和默认模式下是一样的： 
test

注释

Lua中，注释以--[[开始，以--]]结束，例如： 
--[[ my first program in Lua --]]

命名规范

Lua程序中，变量名、函数名等只能以字母或者下划线_开始，后面可以包含数字，但不能包含标点符号，例如@，$，%。Lua是大小写敏感的。

保留的关键字

以下关键字是Lua保留的，不能用做变量名函数名等标识：

and break do else
elseif end false for
function if in local
nil not or repeat
return then true until
while
